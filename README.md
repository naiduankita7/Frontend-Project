The tech stack for the **Frontend-Project** includes:

* **HTML5**: Used to structure the core content for pages such as the home, about, blog, and contact sections.
* **CSS3**: Custom styles are defined in `style.css` to manage layout, typography, and responsiveness.
* **Google Fonts**: The project utilizes external fonts, specifically 'Poppins', for a modern look.
* **Font Awesome**: Vector icons are integrated via CDN to enhance the user interface.
